# Apps Script 업데이트 가이드 (자동 업로드용)

자동 업로드 버튼(`☁️ 재료목록 자동 업로드`)이  
`unknown action: syncIngredients` 에러를 내면,  
웹앱에 배포된 Apps Script 코드가 구버전이라는 뜻입니다.

아래 순서대로 **한 번만** 업데이트하면 됩니다.

## 1) 코드 복사
- 이 저장소의 `apps_script_v5.gs` 파일 내용을 전체 복사합니다.

## 2) Apps Script 열기
- 브라우저에서 `script.google.com` 접속
- 현재 냉털 앱이 쓰는 Apps Script 프로젝트를 엽니다.

## 3) 코드 교체
- 기존 코드 전체 삭제
- 방금 복사한 `apps_script_v5.gs` 내용을 붙여넣기
- 저장(Ctrl/Cmd + S)

## 4) 웹 앱 재배포
- 우측 상단 `배포` 클릭
- `배포 관리` 선택
- 웹 앱 배포를 **새 버전으로 업데이트** (또는 새 배포 생성)
- 접근 권한/실행 주체는 기존 설정 유지

## 5) 앱에서 재시도
- 앱 관리자 모드로 이동
- `☁️ 재료목록 자동 업로드` 버튼 다시 클릭

---

## 빠른 체크
- 여전히 실패하면:
  1) `COMMUNITY_SCRIPT_URL`이 최신 배포 URL인지 확인
  2) Apps Script 배포가 웹 앱 형태인지 확인
  3) 배포 버전이 실제로 업데이트되었는지 확인
