# 냉털 출시 운영 런북 (초간단)

출시 직전에 "문제 생겼을 때 어떻게 대응할지"를 빠르게 보기 위한 문서입니다.

## 1) 서비스 상태 점검 순서

1. 앱 접속이 되는지 확인 (`index.html` 로딩)
2. 레시피 목록이 보이는지 확인 (Google Sheets 연동)
3. 커뮤니티 읽기/쓰기 되는지 확인 (Firebase RTDB)
4. 관리자 기능(승인/거절)이 동작하는지 확인

## 2) 자주 발생하는 장애와 대응

### A. 레시피가 안 뜰 때
- 증상: 첫 화면이 비거나 레시피 수가 0개
- 확인:
  - Google Sheets 공개 상태/권한
  - 시트 탭 이름(`레시피`, `재료목록`) 오타 여부
  - 네트워크 차단 여부
- 임시 대응:
  - 앱 새로고침
  - 캐시 삭제 후 재접속
  - 관리자 공지: "일시적 데이터 동기화 지연"

### B. 커뮤니티 업로드/좋아요가 안 될 때
- 증상: 글 등록 실패, 좋아요 반영 안 됨
- 확인:
  - Firebase Rules 최근 변경 사항
  - Firebase Auth 익명 로그인 상태
  - RTDB 리전/프로젝트 상태
- 임시 대응:
  - 관리자 공지: "커뮤니티 기능 점검 중"
  - 승인 작업은 일시 보류

### C. 관리자 승인 버튼이 안 될 때
- 증상: 승인/거절 클릭 시 반응 없음
- 확인:
  - `adminUids` 등록 여부
  - 로그인 UID 일치 여부
  - 브라우저 localStorage 초기화 여부
- 임시 대응:
  - 관리자 재로그인
  - 새 탭에서 관리자 페이지 재접속
